Welcome to Chemistry Js

What is Chemistry Js

Chemistry Js is a web app made to help students understand and calculate chemistry concepts.

What can Chemistry Js do

As of the latest beta (beta5), Chemistry Js can solve equilibrim related problems using both initial concentrations and final concentrations and it can generate the correct IUPAC name of a biomolecule given some informations about the molecule.
